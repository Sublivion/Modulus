About

Modulus is a discord bot in development by CalvinTLincoln and Sublivion (SUB#2251). It will have web control and logs, as well as some awesome commands and features.


Getting Started

To get started, you must first connect your discord account

Then, invite the bot to your server using the invite button below.

Set up your server using the Control page

You can view your logs using the Logs page

If at any point you need a reminder of the commands, hit the Commands button or
say -help
